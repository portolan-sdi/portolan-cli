## Why GitHub Issues?

- **Single source of truth** — No manual checkmarks to maintain
- **Automatic status** — Issues close when PRs merge
- **Labels** — Filter by `roadmap:mvp`, `roadmap:next`, `roadmap:future`
- **Milestones** — Track progress toward releases
