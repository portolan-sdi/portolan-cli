## Why This Change?

Manual ROADMAP.md maintenance doesn't scale:

- Checkmarks drift from actual implementation status
- No link between roadmap items and implementation work
- Duplicate tracking (roadmap vs issues vs PRs)

GitHub Issues provide:

- Single source of truth
- Automatic status tracking via PR links
- Milestone-based progress visualization
- Labels for filtering by phase/priority
